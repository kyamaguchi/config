## cdto (Shortcut for Terminal from Finder)

<http://code.google.com/p/cdto/>

### Install

    Download
    Extract
    Copy to /Application
    Execute .app
    Drag and drop cdto.app on sidebar
    
    Drag and drop any file/directory to cdto shortcut on sidebar
